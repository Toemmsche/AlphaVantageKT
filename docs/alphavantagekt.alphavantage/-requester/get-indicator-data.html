fun getIndicatorData(symbol: String, securitySymbol: String, interval: IndicatorInterval, params: Map<String, String>): MutableList<IndicatorQuote>

Returns historical data of an indicator for an underlying security.

Parameters

symbol - The indicator symbol, e.g. SMA (Simple Moving Average).

securitySymbol - The ticker of the underlying security, e.g. AAPL, USDEUR or BTC

interval - The interval between datapoints.

params - A map of parameters for the api call, some of which might be required for certain indicators
